# GE SS Two Font Arabic

This directory should contain the GE SS Two Font Arabic font files.

## Required Font Files

Place the following font files in this directory:

- `GE-SS-Two-Regular.otf` or `GE-SS-Two-Regular.ttf` (Regular weight)
- `GE-SS-Two-Bold.otf` or `GE-SS-Two-Bold.ttf` (Bold weight)
- `GE-SS-Two-Light.otf` or `GE-SS-Two-Light.ttf` (Light weight - optional)
- `GE-SS-Two-Medium.otf` or `GE-SS-Two-Medium.ttf` (Medium weight - optional)

## Font Format

The project supports both `.otf` (OpenType) and `.ttf` (TrueType) formats. The CSS will automatically use the available format.

## Note

GE SS Two is a commercial font. Ensure you have the proper license to use it in your project.
